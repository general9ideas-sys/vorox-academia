# VOROX Academia — Landing Page

Landing page de alto impacto inspirada en la estructura de [SoyHenry](https://www.soyhenry.com), con identidad visual VOROX.

## Manual de Marca (Agencia Priza)

Basado en `Manual de Marca VOROX - Agencia Priza.pdf`.

| Color | HEX | Pantone | Uso |
|-------|-----|---------|-----|
| Azul principal | `#385EA7` | 2132 C | Dominante: fondos, textos principales |
| Amarillo principal | `#FDC700` | 1235 C | Exclusivo: CTAs y ofertas |
| Gris secundario | `#B3B3B3` | Cool Gray 7 C | Textos secundarios, bordes |
| Crema | `#FFF2C1` | 7401 C | Fondos de apoyo (uso moderado) |

**Tipografía:** Montserrat Bold (títulos) · Raleway Regular (cuerpo).

**Logotipo:** versión a color en fondos claros (`assets/logo-light.svg`), versión negativa en fondos oscuros (`assets/logo-dark.svg`). Área de protección aplicada en navbar y footer.

## Estructura de secciones

1. **Hero** — Título, propuesta de valor, CTA amarillo, métricas
2. **Garantía** — Inversión sin riesgo (fondo crema)
3. **Carreras** — Tarjetas con duración, modalidad y "Ver más"
4. **¿Por qué VOROX?** — Pilares del método (fondo azul)
5. **Testimonios** — Muro del amor con egresados
6. **Formulario** — Cierre con fondo azul para aplicar

## Cómo ver la página

Abrir `index.html` en el navegador, o servir con un servidor local:

```bash
# Python
python -m http.server 8080

# Node (npx)
npx serve .
```

Luego visitar `http://localhost:8080`.
